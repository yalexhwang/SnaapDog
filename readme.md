# SnaapDog 

SnaapDog enables an open collaboration between seekers, finders, and shelters to get roaming dogs home.
- Lost a dog? create an alert! post a photo of your dog, add description by tags for easy matching, set the radius for neighborhood alert
- Found a dog? create a sighting! post a photo of the dog, add description by tags for easy mathching, set the found location
- Connect with each other without revealing personal contact information
